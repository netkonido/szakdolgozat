\section{Specifikáció}

A feladat egy webalkalmazás fejlesztése ami személyre szabott önéletrajzokat készít a felhasználónak. A program lehetőséget nyújt a felhasználónak adatai beviltelére szöveges mezőkön, file feltöltésen, illetve LinkedIn profil megadásán keresztül. A program generatív mesterséges intelligencia és Retrieval-Augmented Generation segítségével feldolgozza a felhasználó által megagott adatokat és fileokat, majd egy személyre szabott önéletrajzot készít. A kész önéletrajzot a felhasználó áttekintheti, szükség esetén újrageneráltathatja, illetve több formátumban letöltheti. A felhasználó akármikor visszatérhet az előző lépéshez, vagy megszakíthatja a munkamenetet. Minden adat amit a felhasználó megadott, törlésre kerül a programból való kilépés után, valamint az aktuális munkamenet megszakítását követően.

\subsection{Funkcionális követelmények}

\paragraph{Megnyitás:}

A program webalkalmazás formájában legyen elérhető modern webböngészőből\footnote{Google Chrome, Microsoft Edge, Safari, Mozzilla Firefox, Opera, Brave}. A böngészőben a [megfelelő elérési útvonal] keresősávba való beírása után jelenjen meg a nyitólap és automatikusan induljon el egy új munkamenet.

\pagebreak

\paragraph{Adatbevitel:}

A felhasználó tudjon több módon adatokat bevinni.

\begin{enumerate}
    
    \item LinkedIn hivatkozás megadásával

A felhasználó tudjon LinkedIn fiók hivatkozást megadni. A program mesterséges intelligencia segítségével automatikusan ismerje fel és töltse be a hivatkozás által mutatott profil oldalán található adatokat.

    \item File feltöltéssel
    
A felhasználó tudjon fileokat feltölteni. A feltöltött fileok lehetnek szöveges vagy kép formátumban, és tartalmazhatnak korábbi önéletrajzokat, okleveleket, elismervényeket, bizonyítványokat és más tanúsítványokat. A program mesterséges intelligencia segítségével automatikusan ismerje fel és töltse be a dokumentumokban található releváns információkat. 
    
    \item Szöveges mezőn keresztül

A felhasználó tudjon manuálisan adatokat megadni. A bevitt szöveges adatok tartalmát a felhasználó tudja kategorizálni legördülő listából való kiválasztással, vagy egyéb esetben kézi megadással. 

\end{enumerate} 

\paragraph{Állás specifikálása:}

\begin{enumerate}
    
    \item Hivatkozással: 
    
    A felhasználó tudja megadni az álláshirdetéshez mutató hivatkozást. A program mesterséges intelligencia segítségével automatikusan ismerje fel és töltse be az álláshirdetésben szereplő követelményeket és releváns információkat. A felhasználó tudja szerkeszteni a mesterséges intelligencia által kinyert információkat.
    
    \item Manuálisan:

    A felhasználó tudja megadni az álláshirdetés részleteit szöveges mezőben való kézi bevitelen keresztül.

\end{enumerate}

\paragraph{Visszalépés:}

A felhasználó tudjon minden lépés után visszalépni az előző oldalra.
A program töltse vissza az előző oldalakon megadott információkat.

\paragraph{Generálás:}

Az program generáljon mesterséges intelligencia segítségével egy önéletrajzot a felhasználó által megadott adatok felhasználásával. A felhasználó tudjon kérést leadni a generálás megismétlésére.

\paragraph{Letöltés:}

A felhasználó tudja letölteni az önéletrajzot. A felhasználó tudja kiválasztani a formátumot, amiben letölti az önéletrajzot.

\paragraph{Kilépés}

A felhasználó tudjon kilépni az alkalmazásból a böngészőablak bezárásával. A felhasználó tudja megszakítani a munkamenetet a folyamat akármelyik lépése közben. A program lépjen vissza a kezdőképernyőre a munkamenet megszakítása után.

\subsection{Nem funkcionális követelmények}

\paragraph{Felhasználók:}

A felhasználó tudja bejelentkezés vagy regisztráció nélkül használni az alkalmazást. Az alkalmazásnak ne legyen adminisztrátori felülete.

\paragraph{Munkamenet:}

Az program lássa el egyedi azonosítóval az új munkameneteket. A program az azonosító segítségével kövesse a felhasználó által megadott adatokat és feltöltött fileokat. A program szakítsa meg a munkamenetet 10 perc inaktivitás után.

\paragraph{Adatkezelés:}

A program ne tárolja perzisztensen a felhasználó által megadott adatokat és feltöltött fileokat. A program elindításakor az adatbázis inicializálódjon üres állapotba. Az adatbázisban csak az aktív munkamenetekhez köthető adatok kerüljenek tárolásra. A munkamenet megszakítása után a munkamenethez kapcsolódó tárolt adatok törlődjenek.

\section{Tervezés}

\subsection{Felhasználói felület terv}

\begin{enumerate}
    \item \textbf{Nyitólap:}

Ez az oldal jelenik meg amikor a felhasználó megnyitja a webböngészőn keresztül az alkalmazást. Ezen az oldalon egy gomb van "Kezdés" felirattal.
    
    \item \textbf{Adatbevitel file feltöltéssel, LinkedIn hivatkozással:}

    Ezen az oladlon a felhasználó egy szövegdobozban megadhatja a LinkedIn profiljának linkjét vagy tölthet fel fileokat [drag-and-drop módszerrel vagy] a "File feltöltés" gomb megnyomásával, majd a megfelelő file kiválasztásával. A feltöltött fileok nevei egy listában jelennek meg. Legfeljebb 10 file feltöltésére van lehetősége a felhasználónak, ezt egy számláló jelzi a gombon.  Ha a felhasználó befejezte az adatbevitelt, a "Tovább" gombbal léphet a következő oldalra.
    
    \item \textbf{Adatok áttekintése és szerkesztése:}

    Az oldal megjelenésekor még nem biztos, hogy a mesterséges intelligencia modell visszatért a kinyert adatokkal. Az adatok beérkezéséig az oldalon egy betöltési animáció jelenik meg. Az adatok listázásra kerülnek kategóriákba rendezve:

    \begin{enumerate}
        \item Tanulmányok
        \item Munkatapasztalat
        \item Képesítések
        \item Egyebek
    \end{enumerate}

    A felhasználó minden elemnél a szerkesztés ikonra kattintva megváltoztathatja a mező tartalmát. A felhasználó a törlés ikonra kattintva eltávolíthatja az elemet. A felhasználó a hozzáadás ikonra kattintva új elemet hozhat létre. Az új elem tartalmát a megjelenő szövegdobozba való bevitellel adhatja meg, kategóriáját a legördülő menüből választhatja ki. A bevitelt véglegesítheti a zöld pipára kattintva, elvetheti a piros "X"-re kattintva. Ha a felhasználó elégedett az adatokkal, a "Tovább gombra kattintva folytathatja a folyamatot.

    \item \textbf{Álláshirdetés megadása:}

    A felhasználó szöveges mezőn keresztül megadhatja az álláshirdetés részleteit, vagy az álláshirdetésre mutató hivatkozást. A hivatkozás megadása esetében a program mesterséges intelligenciával kinyeri a releváns információkat, és betölti a szöveges mezőbe, amit a felhasználó szerkeszthet a továbblépés előtt.
    
    \item \textbf{Önéletrajz áttekintés:}

    Az önéletrajz vázlat generálása után a felhasználó áttekintheti azt, és az "Újragenerálás" gombbal megismételheti a generálási folyamatot.
    
    \item \textbf{Letöltés:}

    Amennyiben a felhasználó elégedett az eredménnyel, kiválaszthatja a kimeneti formátumot, majd letöltheti az önéletrajzát.
    
\end{enumerate}
